Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Результат: слайс b {77,78,79}

Обьявление слайса включает в себя наименьший индекс и не включает наибольший.
...

```
